# File Format 

部分几何文件类型的格式梳理



### `.Poly`

#### 组成部分

* **Node List**:
  * 网格的顶点
  * 格式：
    * 第一行：
      * 8 3 0 0
      * 节点数量 + 空间维度 + 节点属性 + 边界标记
    * 其余行：
      * 1 50 0 0
      * 节点序号 + 3d坐标
* **Facet List**:
  * 网格的面/多边形
  * 格式：
    * 第一行：
      * 6 0
      * 面的数量 + 边界标记
    * 其余组：
      * 面上有多少个多边形
      * 多边形有几个顶点
      * 每个顶点的index
* **Hole List**:
  * 网格内部的孔洞
* **Region List**:
  * 网格中部分区域的材料等信息
  * 格式：
    * 第一行：
      * 区域数量
    * 剩余行：
      * 25 25 0.5 1
      * 区域内部一个点坐标 + 区域标号
